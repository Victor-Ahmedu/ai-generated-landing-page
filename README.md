**3MTT Software Development Track | Mini Project 1**
**Student:** Victor Ahmedu

## What I Built

A responsive, one-page marketing landing page for **FlowDesk**, a fictional productivity app. The page includes four sections: Hero, Features, Call to Action (CTA), and Footer.

## How AI Helped

I used an AI assistant to handle the **planning** phase of this project:
- Suggested the app name (FlowDesk) and one-line value proposition
- Drafted the hero headline and subtext options
- Generated six feature card ideas with short descriptions
- Recommended the colour palette (deep navy + accent blue) and the Inter font pairing
- Suggested using a 3-column CSS Grid for the features section and Flexbox for the hero/navbar

I wrote **all the HTML and CSS code myself**. The AI output was used only as a design brief and content plan, not as final code.

## Project Structure

```
ai-generated-landing-page/
├── index.html    — Page structure using semantic HTML5
├── style.css     — All styling: Flexbox, Grid, and media queries
└── README.md     — This file
```

## Key Techniques Used

- **Semantic HTML5** — `<header>`, `<section>`, `<article>`, `<footer>`, `<nav>`
- **CSS Flexbox** — Navbar layout, hero buttons, footer links
- **CSS Grid** — Feature cards in a 3-column grid
- **CSS Custom Properties (Variables)** — Consistent colour and spacing
- **Media Queries** — Grid collapses to 1 column on screens ≤ 768px
- **Responsive viewport meta tag** — Ensures correct rendering on mobile

## How to View

1. Open the `ai-generated-landing-page` folder in VS Code
2. Right-click `index.html` and choose **Open with Live Server**  
   (or simply open `index.html` directly in any browser)
3. To test mobile view: press F12 in your browser → click the device icon
